Autonomous Structural Intelligence System

Architecture Diagram  ·  ASIS Project

Input
🏗 Floor Plan Image (PNG / JPG)
Uploaded by architect, engineer, or construction company
↓
Stage 1
🔍
OCR Detection
stage1/parser/ocr_detector.py
EasyOCR reads room labels and dimensions from the image. Regex normalizes variations — "BED ROOM", "BEDRM" → "bedroom". Filters below 20% confidence.
→ Room counts, dimensions, annotations
Stage 2
📐
Vision Analysis
stage2_3/image_processor.py
stage2_3/graph_builder.py
OpenCV pipeline: Grayscale → Otsu threshold → Morphological cleanup → Canny edges → HoughLinesP. Graph builder snaps nodes (15px), classifies junctions and wall types.
→ Wall graph: outer / spine / partition
Stage 3
🧊
3D Model
try/try_app.py
try/vision_parser.py
Upgraded Three.js exporter. Pixel → meters (1px = 5cm). BoxGeometry per wall, TWEEN camera flyto, click-to-select, furniture (beds, TV units), columns, tooltips.
→ Self-contained HTML 3D viewer
Stage 4
🧱
Material Analysis
stage4/tradeoff_engine.py
stage4/Book1.xlsx
Weighted tradeoff formula: score = (w_s×strength + w_d×durability) / (w_c×cost). Different weights per element type — columns prioritise strength (0.65), partitions prioritise cost (0.60).
→ Top 3 materials per element + scores
Stage 5
🤖
AI Report
stage5/prompt_builder.py
stage5/gemini_client.py
Structured prompt injects actual wall counts, material scores, and weights. Gemini 2.5 Flash generates 5-section engineering report. Rendered with styled headings + markdown. Downloadable as .txt.
→ 5-section structural engineering report
Stage 6
⛓
Blockchain Audit
web3/contracts/lib.rs
web3/app.py (Flask + Stellar SDK)
Soroban smart contract on Stellar testnet. SHA-256 floor hash + materials + owner wallet stored immutably. Flask backend signs and broadcasts transactions. 4-tab Streamlit UI.
→ On-chain audit certificate + ownership
